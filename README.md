# 小小攝影師（PhotoLession）

**小朋友與大人都能輕鬆學攝影**的手機互動學習 Web App。用生活比喻、真實照片與動手互動，把「對焦、光線、構圖、光圈快門 ISO」從零講到懂——不管你是想帶孩子入門的家長，還是想把手機拍好的大人，都能一課一課玩下去。

🔗 **線上試玩**：https://photolession.vercel.app
（純靜態、免安裝、免註冊；用瀏覽器打開即可用，**推薦 iPad**）

<br>

## ✨ 特色

- **37 堂課、4 條學習軌**：A 相機操作 6 課／B 光線曝光 9 課／C 構圖 12 課／D 生活實戰 6 課，每軌再加一關「驗收綜合關 🏁」，循序解鎖。
- **20 種互動工具，邊玩邊懂**：拖太陽看光影、拉光圈滑桿玩景深、快門定格瞬間、三分格線構圖、色環配色……觀念用手指玩一次就記住。
- **191 張真實照片＋58 幅量身示意圖**：教學照片全部精選自 Pexels（175 位攝影師），另有 53 幅專為每個觀念手繪的概念圖＋5 個手機介面模擬，一頁一圖不含糊。
- **📸 動手拍作業（33 個）**：每課附一個真實拍攝任務，孩子拍完存進裝置，家長區審核通過該課才點亮。
- **✏️ 隨堂小考（159 題）**：每課數題選擇 / 複選，答錯有解說，觀念當場鞏固。
- **🔊 語音朗讀**：每段課文都能點喇叭唸出來（繁體中文），還不太會認字的小朋友也能自己上課。
- **💡 趣味小知識（33 篇）**：每課補一篇攝影背後的故事與原理。
- **看得懂的說明**：一頁一觀念、真實參數（f/1.8、1/1000s、ISO 3200）、名詞卡＋跨課連結，不用背術語。
- **進度看得見**：每課「學習📖 / 測驗✏️ / 作品📸」三盞燈與完成百分比，集滿得 🏅，全破關有畢業祝賀 🎓。
- **🙋 小朋友求救鈕**：卡關可一鍵留言給家長，自動附上當下課程情境。
- **📱 同時支援 iPhone 與 iPad**：介面會依螢幕自動調整版面，**推薦用 iPad**（大螢幕玩互動、看照片最舒服）。

## 📱 支援裝置

本程式目前**只在 iPhone 與 iPad（Safari）上測試過**，並針對這兩種裝置最佳化，推薦使用 **iPad**。它是純網頁 App，理論上其他手機、平板或桌機瀏覽器也能開，但未經測試，不保證顯示與互動完全正常。

## 🔒 隱私

這個 App 沒有後端、沒有帳號、沒有廣告、沒有任何追蹤或分析工具。

- 孩子拍的照片只存在**這台裝置**（瀏覽器 IndexedDB），永不上傳。
- 學習進度只存在**這台裝置**（localStorage）。
- 不收集任何個人資料。

> 附註：程式裡有一個「回報問題給開發者」的小功能，是**專門留給開發者自己除錯用的**，需要密碼才能打開，一般使用者看不到、也用不到它。
>
> 就算它被打開，送出的也只有你**手動打的那幾句問題描述**——**孩子的照片、學習進度完全不會被送出**，隱私一樣安全。

## 🗂 專案結構

- `app/`——兒童 App（Vite ＋ React ＋ TypeScript ＋ Tailwind，純靜態，部署於 Vercel）
- `content/`——課程內容庫（課程 JSON ＋ 教學照片 ＋ 標註，含每張照片的授權與作者紀錄）
- `pipeline/`——素材管線（Pexels 抓圖／去重／衍生示意圖，Python）
- `docs/course-overview.md`——課程總覽

## 🛠 開發

```bash
# App 開發／建置（prebuild 會自動把 content/ 同步到 public/content）
npm --prefix app run dev
npm --prefix app run build

# 素材管線（需自備 Pexels API key，放在 pipeline/.env）
python pipeline/fetch.py --concept all
```
